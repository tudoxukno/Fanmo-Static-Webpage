Importing

First step for using a effect from this pack is very simple. You need to import aero-style.css file or compressed aero-style.min.css file, located on "CSS" folder, on your project. You also have option to import SCSS files, located on "SASS" folder, and compile this.

Classes

All effects have a class. This is very important, because without this you can not acces the effect. Is important to know class name for a preffered effect, you can check the name of an effect on index.html from "aero-effects-demo" folder ( "Demo - No Images" ).

Implementing


<div class="class-effect direction-class axis-class">
	<div class="effect-wrapper">

		<!-- Image Box -->
		<div class="img-box">
			<img src="Your image link here." alt="" />
		</div>

		<!-- Image Box -->
		<div class="info-box">
			<div class="info-content">
				<h3>Your Title</h3>
				<p>Lorem ipsum dolor sit amet, consectetur
				adipisicing elit. Enim, quam dignissimos
				sed cupiditate suscipit.</p>
			</div>
		</div>

	</div>
</div">

On code from the right you can see an HTML5 structure for using a effect from "Aero | Gleesik™ CSS3 Hover Effects". If you create this structure and add a class for an effect + direction or axis ( this is very important for effects with multiple directions or axis ), all of your content was implemented on choosed effect.


First of all check the class name for your prefered effect and change class-effect with class of prefered effect.


That's all! Now you can use all effect from this pack! I hope you like this.
